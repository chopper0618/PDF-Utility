# Sprint 0.8.4: Preview keyboard navigation

## Issue

- Issue #24: Preview keyboard navigation

## 目的

サムネール拡大プレビューを開いたまま、前後ページを確認できるようにする。

## 実装内容

- プレビュー中の `ArrowLeft` / `ArrowRight` に前後ページ移動を追加
- プレビュー下部に「前へ」「次へ」ボタンを追加
- 先頭・最終ページでは移動できない方向のボタンを無効化
- 前後移動時に選択ページを同期
- 前後移動時も大きめのプレビュー画像を再生成
- プレビュー中の Delete / Backspace / Undo / Redo 誤爆防止を維持

## UX 判断

左右キー移動は、プレビュー中だけに限定する。通常画面では既存ショートカットやブラウザ操作との衝突を避ける。

「前へ」「次へ」ボタンも配置し、キーボード操作に慣れていない利用者にも移動方法が分かるようにする。

## 確認項目

- サムネールをダブルクリックしてプレビューを開ける
- 右矢印キーで次ページへ移動できる
- 左矢印キーで前ページへ移動できる
- 先頭ページで左矢印を押しても壊れない
- 最終ページで右矢印を押しても壊れない
- 「前へ」「次へ」ボタンで移動できる
- プレビュー中に Backspace / Delete でページが削除されない
- プレビューを閉じた後、削除・複製・回転・移動・並び替えが通常どおり動く
- Page Map のクリックジャンプ・ドラッグ並び替えが壊れていない
- PDF作成が壊れていない
