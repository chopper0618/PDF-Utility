# Issue #23: Thumbnail preview overlay

## 背景

サムネール一覧はページ構成を素早く把握するには有効だが、ページ内容を確認するには表示が小さい。
PDF整理作業では「このページで合っているか」を確認してから削除・移動・複製する場面が多い。

## 目的

中央サムネールをダブルクリックしたときに、ページを大きくプレビュー表示できるようにする。

## Scope

- サムネールカードのダブルクリックでプレビューを開く
- 背景を暗くし、中央に拡大プレビューカードを表示する
- 元PDFからプレビュー用の大きめ画像を生成し、サムネール画像の単純拡大に見えないようにする
- プレビューには現在ページ番号、ファイル名、元ページ番号、回転角度を整理して表示する
- ×、背景クリック、Escape で閉じる
- プレビュー表示中は Delete / Backspace / Undo / Redo / Select All などのページ操作ショートカットを実行しない

## Out of Scope

- 左右キーで前後ページへ移動
- 拡大率変更
- プレビュー上での回転・削除・複製

## Acceptance Criteria

- サムネールをダブルクリックすると、サムネールより大きいプレビューが表示される
- プレビュー表示中に Escape で閉じられる
- 背景クリックと × ボタンで閉じられる
- プレビュー表示中に Backspace / Delete でページが削除されない
- プレビューを閉じたあと、既存のページ操作が通常どおり動く
- Page Map、Undo / Redo、PDF作成が壊れていない
