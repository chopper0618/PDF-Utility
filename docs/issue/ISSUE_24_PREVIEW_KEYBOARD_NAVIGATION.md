# Issue #24: Preview keyboard navigation

## 背景

Sprint 0.8.3 で、サムネールをダブルクリックして拡大プレビューを開けるようになった。

ただし、複数ページを確認したい場合は、一度プレビューを閉じてから別ページを開き直す必要がある。ページ内容を連続確認する用途では手数が多い。

## 目的

拡大プレビューを開いたまま、前後ページへ移動できるようにする。

## 方針

- プレビュー中だけ `ArrowLeft` / `ArrowRight` を前後ページ移動に割り当てる
- フッターに「前へ」「次へ」ボタンも配置する
- 先頭ページでは「前へ」を無効化する
- 最終ページでは「次へ」を無効化する
- プレビュー中の Delete / Backspace / Undo / Redo 誤爆防止は維持する
- プレビュー移動時は Page Map / Page Viewer の選択状態も同期する

## 今回やらないこと

- プレビュー内でのズーム変更
- プレビュー内での回転操作
- プレビュー内での削除・複製
- PDFビューアのような連続スクロール表示

## 受け入れ条件

- プレビュー中に右矢印キーで次ページへ移動できる
- プレビュー中に左矢印キーで前ページへ移動できる
- 先頭ページで左矢印を押しても壊れない
- 最終ページで右矢印を押しても壊れない
- フッターの「前へ」「次へ」ボタンで移動できる
- プレビュー中に Backspace / Delete でページが削除されない
- プレビューを閉じた後、通常のページ操作が壊れていない
