# Design System

## Concept

Professional / Simple / Fast.

## Buttons

- Ribbon button: 54 × 58 px
- Thumbnail action: 30 × 30 px
- Danger action: delete only
- Icon: Material Symbols

## Thumbnail Card

- ページ番号バッジを左上に表示
- 操作ボタンはホバー・選択時に表示
- 選択中は青枠とリングで表示
- 複製ページはタグ表示

## Spacing

- 4 / 8 / 12 / 16 / 24 / 32 px

## Status

- Hover: light primary
- Selected: primary ring
- Drop target: success ring
- Danger: red
