# TERMS

PDF Utility で使う用語を統一する。

## 主要用語

| 用語 | 意味 |
|---|---|
| PDF Utility | PDF の結合・ページ整理を行う Web アプリ |
| Page Viewer | 中央またはメイン領域のページ表示エリア |
| Page Map | 左側のページ構成一覧。旧称「ファイルツリー」「ページ一覧」 |
| Page Order | 現在のページ順を保持する内部状態 |
| Page Item | 1ページ分の表示・操作単位 |
| Selected Page | 現在選択中のページ |
| Drag Reorder | ドラッグ操作によるページ並び替え |
| Move Command | ボタンやメニューによるページ移動操作 |
| Status Bar | 読み込み状態、ページ数、操作結果を表示する下部領域 |
| Ribbon | 上部の主要操作エリア |
| Properties | 選択ページやファイル情報を表示・調整する領域 |

## Page Map について

左側のページ一覧／ファイルツリーは、今後 **Page Map** と呼ぶ。

Page Map は単なるファイル一覧ではなく、以下の役割を持つ。

- PDF 全体のページ構成を表示する
- 並び替え・削除・移動後の状態を反映する
- クリックで該当ページへジャンプする
- 現在選択中のページを示す
- 大量ページ PDF で全体の見通しを助ける

## 避けたい呼び方

| 避けたい呼び方 | 理由 |
|---|---|
| ファイルツリー | ページ単位の構成を表すため、ファイルだけではない |
| 左の一覧 | 役割が曖昧になる |
| サイドバー | UI 領域名としては使えるが、機能名としては Page Map を優先 |

## Page Order と表示の関係

Page Order は内部状態。Page Viewer と Page Map は Page Order をもとに描画される。

操作後に Page Viewer だけ更新し、Page Map が古いままになる状態は禁止。
