## 残っている UX 課題

### 1. 移動ダイアログの独自モーダル化

現在の「移動...」はブラウザ標準 `prompt()` を使っているため、ダイアログタイトルにローカルアドレスが表示される。

機能としては動作するが、毎日使う業務用ツールとしては完成度が低く見えるため、独自モーダル化を次候補にする。

### 2. ページ操作 UX 安定化

- 複数ページ選択の扱い整理
- 削除後の選択状態整理
- 移動後のスクロール位置制御
- ページ番号表示の統一
- 操作後の軽いフィードバック表示
