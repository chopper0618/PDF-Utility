# Requirements

## v1.0 scope

PDF Utility is a browser-based PDF page organization tool for hospital workflows.

## Core requirements

- Local PDF import
- Multiple PDF import
- Drag & drop PDF import
- Thumbnail display
- File tree display
- Page selection
- Page rotation
- Page deletion
- Page duplication
- Page reordering
- Undo / Redo
- PDF export and download

## Privacy requirement

PDF files should be processed in the browser whenever possible. The application should not upload PDFs to external servers for normal editing operations.

## Current sprint requirement

v0.2.0-alpha implements PDF import and thumbnail display only. Editing and export are planned for later sprints.
