# RIBBON

## 役割

Ribbon は、PDF Utility の主要操作を配置する上部 UI 領域。

## 想定する操作

- PDF を追加
- PDF を書き出し
- 選択ページを削除
- 選択ページを指定ページへ移動
- 選択ページを先頭へ移動
- 選択ページを末尾へ移動
- 表示倍率や表示方法の切り替え

## Sprint 0.7.0 で追加・確認する操作

### 移動...

選択ページを、指定したページ番号へ直接移動できるようにする。UI 表記は短く「移動...」とする。

例。

- 現在 8 ページ目のページを選択して「3」と入力すると、3 ページ目へ移動する
- 複数ページ選択時は、選択ページ群をまとまりとして移動し、入力したページ番号を移動後の先頭位置として扱う

## UI 方針

- 「上へ移動」「下へ移動」は廃止する
- Ribbon には「移動...」「先頭へ」「末尾へ」を置く
- 今回は標準 `prompt()` でよいが、ブラウザ標準ダイアログではタイトルにローカルアドレスが表示される
- 将来的には独自モーダルに置き換える

## 注意点

- 選択ページがない場合は移動ボタンを無効化する
- 1未満、総ページ数超過、数値以外は移動しない
- 移動後は Page Viewer と Page Map を同期する
- Status Bar に移動結果を表示する


## 残課題

現在は標準 `prompt()` を使用しているため、ダイアログタイトルにローカルアドレスが表示される。完成度を上げるため、Sprint 0.7.1 以降で独自モーダル化する。
