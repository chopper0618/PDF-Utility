# PROPERTIES

## 役割

Properties は、選択中ページや PDF ファイルに関する補助情報を表示する領域。

現時点では Sprint 0.7.0 の主対象ではないが、Page Map や Page Viewer と選択状態を共有する可能性がある。

## 表示候補

- 選択中ページ番号
- 元ファイル名
- 元 PDF 内ページ番号
- ページサイズ
- 回転情報
- 操作履歴の簡易表示

## Sprint 0.7.0 での扱い

Properties が未実装または簡易実装の場合、無理に拡張しない。

ただし、選択中ページが変わったときに Properties があるなら更新する。

## 更新タイミング

- Page Viewer でページ選択
- Page Map でページ選択
- 移動コマンド後
- 削除後
- PDF 読み込み完了後

## 注意点

- Page Map と異なるページを表示しない
- 削除済みページの情報を残さない
- 情報が未取得なら空欄または「未取得」と表示する
