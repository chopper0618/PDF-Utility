# STATUSBAR

## 役割

Status Bar は、PDF Utility の現在状態、進捗、操作結果を表示する下部 UI 領域。

## 実装済みとして扱う表示

- PDF 読み込み中のプログレス表示
- 「〇ページ読み込み完了」

## 表示するメッセージ例

| 状況 | メッセージ例 |
|---|---|
| 読み込み開始 | PDFを読み込み中... |
| 読み込み中 | 12ページ読み込み完了 |
| 読み込み完了 | 24ページ読み込み完了 |
| 並び替え | ページを移動しました |
| 移動コマンド | 3ページ目を8ページ目へ移動しました |
| 削除 | 1ページを削除しました。Undoで元に戻せます。 |
| Page Map クリック | 10ページ目へ移動しました |
| エラー | PDFの読み込みに失敗しました |

## Sprint 0.7.2 での削除表示

削除は即時実行とする。確認ダイアログは出さない。

代わりに、削除後は Status Bar で削除数と Undo 可能であることを明示する。

削除は取り消し可能だが破壊的操作なので、通常メッセージより目立つように赤系の表示にする。

```text
1ページを削除しました。Undoで元に戻せます。
3ページを削除しました。Undoで元に戻せます。
```

## 注意点

- メッセージは短くする
- エラーは原因と次の行動がわかるようにする
- 操作が成功したかどうかがわかるようにする
- 削除などの破壊的操作は `danger` 表示で通常メッセージと区別する
