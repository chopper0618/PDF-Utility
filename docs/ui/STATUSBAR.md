# STATUSBAR

## 役割

Status Bar は、PDF Utility の現在状態、進捗、操作結果を表示する下部 UI 領域。

## 実装済みとして扱う表示

- PDF 読み込み中のプログレス表示
- 「〇ページ読み込み完了」

## 表示するメッセージ例

| 状況 | メッセージ例 |
|---|---|
| 読み込み開始 | PDFを読み込み中... |
| 読み込み中 | 12ページ読み込み完了 |
| 読み込み完了 | 24ページ読み込み完了 |
| 並び替え | ページを移動しました |
| 移動コマンド | 3ページ目を8ページ目へ移動しました |
| 削除 | 1ページを削除しました |
| Page Map クリック | 10ページ目へ移動しました |
| エラー | PDFの読み込みに失敗しました |

## Sprint 0.7.0 で追加したい表示

- ドラッグ移動完了時の結果
- 移動コマンド完了時の移動先
- Page Map クリック時のジャンプ先
- 移動できない場合の理由

## 注意点

- メッセージは短くする
- エラーは原因と次の行動がわかるようにする
- 操作が成功したかどうかがわかるようにする
