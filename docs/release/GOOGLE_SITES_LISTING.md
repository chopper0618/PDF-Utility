## Full description

```text
PDF Utility は、日常業務で使いやすいように作成している、ブラウザベースの簡易 PDF ページ編集ツールです。

PDF の読み込み、ページの並び替え、削除、回転、複製、複数 PDF の結合、プレビュー確認、保存ができます。
Adobe Acrobat のような総合編集ソフトではなく、ページ整理を軽く分かりやすく行うことを目的にしています。

使用するときは、Google Sites 内に埋め込まれた画面ではなく、リンク先を開いて全画面のブラウザで操作してください。
推奨ブラウザは Google Chrome です。Microsoft Edge は確認できた環境から順次利用してください。
Safari でも利用できますが、保存時のファイル名・保存先指定に制限があります。Safari で保存した PDF は、ブラウザまたは OS のダウンロード設定に従って保存されます。

PDF はブラウザ内で処理されます。Google Drive 連携、自動アップロード、認証機能はありません。
個人情報を含む PDF を保存・共有する際は、所属先の情報管理ルールに従ってください。
```
