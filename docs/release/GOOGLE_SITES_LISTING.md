## Short description

```text
PDF Utility は、PDF のページ並び替え・削除・回転・複製・結合・保存を行うための簡易ツールです。
Chrome または Edge で開いて使用してください。
```

---

## Full description

```text
PDF Utility は、放射線部門の日常業務で使いやすいように作成している、ブラウザベースの簡易 PDF ページ編集ツールです。

PDF の読み込み、ページの並び替え、削除、回転、複製、複数 PDF の結合、プレビュー確認、保存ができます。
Adobe Acrobat のような総合編集ソフトではなく、ページ整理を軽く分かりやすく行うことを目的にしています。

使用するときは、Google Sites 内に埋め込まれた画面ではなく、リンク先を開いて全画面のブラウザで操作してください。
推奨ブラウザは Google Chrome または Microsoft Edge です。

PDF はブラウザ内で処理されます。Google Drive 連携、自動アップロード、認証機能はありません。
患者情報を含む PDF を扱う場合は、院内ルールに従って取り扱ってください。
```

注意：患者情報を含む PDF の取り扱いは、院内ルールに従ってください。
